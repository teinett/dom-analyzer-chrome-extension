    Understanding DOM Elements & Nodes

    A DOM element refers to an actual HTML tag in the DOM tree, such as <div> or <p>.

    A DOM node includes everything: elements, text nodes, comments, and whitespace.

    Lighthouse considers a page’s DOM size excessive if it exceeds 1,400 nodes. It will begin to throw warnings when a page's DOM exceeds 800 nodes.
